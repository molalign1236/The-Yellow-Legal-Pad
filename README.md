# Yellow Legal Pad

Stacey's site. Financial coaching framed as a quiet conversation
across a kitchen table, not a fintech product.

## Run

```
npm install
npm run dev
```

## Build

```
npm run build
```

Output is fully static HTML in `dist/`. Deploy to Vercel (or any
static host).

## Posture

- Static-first, document-shaped, near-zero runtime JS.
- Reading leads; the runtime stays out of the way.
- Calmness is the product. Optimization energy is rejected.
- Slight asymmetry is healthy; perfect symmetry feels industrialized.
- The success metric is "I unexpectedly feel calmer here," not
  "how beautifully designed."

This repo is intentionally small. Resist additions that would make
it bigger without making the writing land more clearly.
